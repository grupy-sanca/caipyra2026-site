Análise de Dados Geoespaciais

Tutorial de análise geoespacial com Python, abordando dados geográficos, mapas e métodos analíticos para compreender dados georreferenciados e assim, gerar hipóteses.

A análise espacial, ou apenas análise geoespacial, é uma abordagem para aplicar a análise estatística e outras técnicas analíticas a dados que possuem um aspecto geográfico ou espacial. Essa análise normalmente é feita utilizando técnicas de renderização de mapas a partir do processamento de dados espaciais e a aplicação de métodos analíticos a conjuntos de dados terrestres ou geográficos. Neste tutorial, serão apresentados conceitos sobre como utilizar a linguagem de programação python para realizar a análise geoespacial de um conjunto de dados com informações geográficas, o que é essencial para obter uma boa compreensão acerca de dados georreferenciados, além de auxiliar na geração de hipóteses que podem ser extraídas a partir dessa análise.

Quais conhecimentos prévios são necessários para que seja possível acompanhar bem a sua atividade?:

Conhecimentos básicos sobre Python, como criação de variáveis, testes condicionais, laços de repetição, definição de funções e manipulação de dados com Pandas

O que as pessoas que participarem podem esperar aprender na sua atividade?:

Os participantes podem esperar uma apresentação introdutória, aplicada e progressiva sobre análise geoespacial com Python, construída para oferecer uma visão clara dos principais conceitos, ferramentas e possibilidades práticas desse campo. A proposta é partir de um problema motivador real, mostrando por que dados espaciais são estratégicos para compreender fenômenos territoriais, apoiar decisões e gerar hipóteses analíticas consistentes. Ao longo da atividade, serão exploradas bases de dados espaciais abertas, com destaque para fontes públicas que podem ser utilizadas em estudos, projetos e aplicações profissionais. Em seguida, serão apresentados os fundamentos dos dados vetoriais e o uso do GeoPandas para leitura, manipulação e análise de informações geográficas em Python. A exposição também abordará sistemas de coordenadas e projeções, relações e operações espaciais, visualização de dados geoespaciais e criação de mapas interativos. Assim, espera-se que o público desenvolva uma compreensão estruturada sobre como organizar, processar, analisar e representar dados georreferenciados de forma tecnicamente adequada.
